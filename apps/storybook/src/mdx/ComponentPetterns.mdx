# Component Patterns

Usages for Layout, Standalone, and Composite components.

## Layout

Layout components from `@tailor-platform/design-systems` simplify the creation of layouts. They are based on Panda's predefined patterns such as stack, hstack, vstack, wrap, and more. Learn more about [panda's patterns](https://panda-css.com/docs/concepts/patterns#usage-with-jsx).

```tsx
import { Box, Flex } from "@tailor-platform/design-systems";

<Flex
  alignItems="center"
  justifyContent="center"
  p={4}
  bgColor="bg.subtle"
  {...args}
>
  <Box p={4}>Item1</Box>
  <Box p={4}>Item2</Box>
</Flex>
```

## Standalone

Standalone components from `@tailor-platform/design-systems`, featuring self-contained logic and styling, can be easily imported and used directly in the code.

```tsx
import { Badge } from "@tailor-platform/design-systems";

<Badge {...args}>Badge</Badge>
```

## Composite

Composite components combine the component logic from `@ark-ui/react` with the styling of `@tailor-platform/styled-system/recipes`. This decoupling facilitates flexible implementations while ensuring consistency across UI. Learn more about [ark-ui](https://ark-ui.com/) / [panda's recipes](https://panda-css.com/docs/concepts/recipes).

```tsx
import {
  TabIndicator,
  TabList,
  Tabs,
  TabTrigger,
} from "@ark-ui/react";
import { tabs } from "@tailor-platform/styled-system/recipes";

<Tabs className={tabs({})} defaultValue="react" {...props}>
  <TabList>
    <TabTrigger value="react">React</TabTrigger>
    <TabTrigger value="solid">Solid</TabTrigger>
    <TabTrigger value="vue">Vue</TabTrigger>
    <TabIndicator />
  </TabList>
</Tabs>
```
