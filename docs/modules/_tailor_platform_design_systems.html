Module @tailor-platform/design-systems

This package offers Design Systems for managing shared UI components across applications.

Architecture

Tailor Design Systems uses the headless components Ark UI with the style capabilities of Panda CSS.

Tech Stacks

Panda CSS

Panda CSS delivers build-time generated styles, RSC compatibility, multi-variant support. Learn more on the Panda CSS documentation.

Commonly used import paths during application development include:

  • @tailor-platform/design-systems
  • @tailor-platform/styled-system/jsx
  • @tailor-platform/styled-system/css
  • @tailor-platform/styled-system/recipes

Find usage examples for each in our Storybook, accessible here.

Ark UI

Ark UI is a headless library crafted for constructing reusable and scalable Design Systems. Learn more on the Ark UI website. Search for examples within our codebase here.

Lucide

We use Lucide for our iconography. Learn more on the Lucide's website.

References

Tailor Inc.'s Design Systems are designed to offer a cohesive and scalable set of UI components and utilities, ensuring consistency and efficiency across applications. For an extensive guide and overview, refer to our main documentation.
